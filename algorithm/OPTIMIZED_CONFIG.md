### 数据增强增强版
```python
增强列表:
1. 随机亮度调整    (概率70%, 范围0.6-1.4)  ← 增强

8. 饱和度调整      (概率30%, 范围0.8-1.2)   ← 新增
```

### 训练参数优化
| 参数 | 原值 | 优化后 | 说明 |
|-----|------|--------|------|
| batch-size | 128 | **256** | RTX 4090 可支持更大批次 |
| weight-decay | 0.0005 | **0.0003** | 适度正则化，不过度抑制 |
| dropout | 0.5 | **0.3** | 数据增强足够强，减少dropout |
| num-workers | 4 | **8** | 加快数据加载 |

### 学习率调度优化
```python
# 优化前: 热重启 (每epoch更新，震荡大)
scheduler = CosineAnnealingWarmRestarts(T_0=10, T_mult=2)

# 优化后: 预热 + 余弦退火 (每step更新，更平滑)
- 预热阶段: 前3轮线性增加
- 退火阶段: 余弦下降到0
- 更新频率: 每个step更新 (更精细)
```

## 检测模型 (YOLOv8) 优化

### 数据增强优化 (针对车牌场景)
| 参数 | 原值 | 优化后 | 理由 |
|-----|------|--------|------|
| degrees | 5.0 | **3.0** | 车牌角度变化小 |
| translate | 0.1 | **0.05** | 车牌位置相对固定 |
| scale | 0.5 | **0.3** | 车牌大小变化有限 |
| shear | 2.0 | **1.0** | 减少畸变 |
| perspective | 0.0 | **0.0** | 保持不变 |
| flipud | 0.0 | **0.0** | 车牌不会倒立 |

### 训练参数优化
| 参数 | 原值 | 优化后 | 说明 |
|-----|------|--------|------|
| batch | 16 | **32** | RTX 4090 支持更大批次 |
| weight_decay | 0.0005 | **0.0003** | 适度正则化 |
| workers | - | **8** | 数据加载优化 |
| cos_lr | - | **True** | 启用余弦学习率 |

## 常见问题快速诊断

| 现象 | 可能原因 | 解决方案 |
|-----|---------|---------|
| Train Acc ↑, Val Acc ↓ | 过拟合 | dropout↑ 或 weight_decay↑ |
| 损失震荡大 | 学习率过大 | lr × 0.5 |
| 收敛慢 | 学习率过小 | lr × 2 或 batch↑ |
| 显存不足 | 批次过大 | batch_size ↓ |
| 数据加载慢 | workers不足 | num_workers ↑ |
